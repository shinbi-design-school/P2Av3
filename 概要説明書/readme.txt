起動手順
1,xamppのダウンロード。
以下のURLよりxamppをダウンロードください。
URL：https://www.apachefriends.org/jp/index.html

2,ダウンロードしたフォルダにあるxampp-control.exeをダブルクリックして起動ください。

3,XAMPP Control Panelの左から２番目に表示されているModuleとPort(s),Actionを確認ください。
3-1:ModuleにあるApacheとMySQLを起動するために右側にあるstartボタンをクリックしてください。
3-2:クリック後、左側にあるMaduleが緑色になることを確認ください。
3-3:XAMPP Control Panelの下半分に起動に関する履歴が記載されますが、エラーがあれば赤色に記載されます。赤色がないことを確認ください。
3-4:XAMPP Control Panelの左から４番目にPort(s)の記載があります。MySQLのPort(s)が3306になっていることを確認ください。
本プロジェクトはMySQLにつなげるためにポート番号を3306番と指定してつなげています。ポート番号が異なる場合、本プロジェクトは起動いたしません。
3-5:ポート番号が異なる場合、XAMPP Control Panelの右上にあるConfigをクリック⇒Configuration of Control Panelが表示され、その表示の右下にService and Port Settingsをクリックしてください。 
Service Settingsが表示されるため、真ん中あたりにMySQLのタグがあるためクリック⇒画面下にあるMain Portを3306と入力し、saveを押してください。次の画面もsaveを押してください。
3-6:XAMPP Control PanelのModule：MySQLのAdminをクリックしてください。ブラウザ上でpHpMyAdminが表示されたことを確認ください。

※詳細は概要説明書に記載してある「xamppの設定」をご確認ください。

4,MySQLのphpMyAdmin上での操作

4-1:画面の上部にある「インポート」を押してください。
4-2:インポートするファイル「参照」をクリックし、本プロジェクトの中に同封されている、quiz.sqlを選択してください。
次に下部あたりにある「フォーマット特有のオプションで、「値がゼロのものに対してAUTO_INCREMENTを使用しない」のチェックを外してください。
その後、１番右下にある「実行」を押してください。
正常にインポートできた場合、画面の左側にデータベース「quiz」ができています。その中のテーブルには「answertime」「inqueryform」「quiz」「quiz2」「quiz3」「quiz4」「thenumberofcorrectanswer」「usertable」があることを確認ください。

※詳細は概要説明書に記載してある「MySQLの設定phpAdmin」をご確認ください。

5,eclipseへのプロジェクトのインポート
5-1:次のURLよりeclipseの「 Pleiades All in One 」をダウンロードください。年度は2020年以降でお願いします。
URL: https://mergedoc.osdn.jp/

5-2:ダウンロード後、ダウンロードしたファイルの中にあるpleiadesファイルの中にあるeclipseファイルの中のeclipse.exeをダブルクリックし起動ください。
5-3:どのフォルダを読み込むか出てくると思いますのでそのままいじらず実行ください。（選択されているファイルは初期ではworkspaceというフォルダが作られると思います。）
5-4:eclipseをダウンロードしたファイルにworkspaceというフォルダが作成されているのでそのworkspaceというフォルダの中にP2Av3.zipを解凍し、解凍したフォルダの中にあるP2Av3、Serversというフォルダをworkspaceの中にコピー＆ペーストしてください。
5-5:eclipse.exeを起動し、一番左上のファイル（F)をクリック、選択の中のインポートをクリックしてください
5-6:選択画面の中で、一般⇒フォルダーまたはアーカイブからプロジェクト　を選択、次へをクリックしてください。
5-7:ディレクトリをクリックし、workspaceの中へ移したファイル（P2Av3とServers）を選択してください。その後、完了をクリックしてください。
5-8:eclipseの画面上にインポートされたP2Av3のフォルダができているので、それを右クリックして選択の中にある「ビルドパス」にカーソルを合わると選択がでてくるので、その中の「ビルドパス構成」をクリックしてください。
5-9:Javaビルドパスのライブラリーをクリック⇒カーソルをクラスパスに合わせて、ライブラリーの追加をクリック⇒サーバーランタイムを選択、次へをクリック⇒Tomcat9(Java11)を選択、完了をクリックしてください
5-10:プロジェクト・ファセットをクリック⇒ランタイムをクリック⇒Tomcat9（Java11）にチェックが入っているか確認、適用して閉じるをクリック

※詳細は概要説明書に記載してある「eclipseへプロジェクトのダウントード」をご確認ください。

6.サーバーの設定
6-1:一番初めの画面で上にあるウィンドウ（W)をクリック
6-2:ビューの表示をクリック
6-3:サーバーをクリック
6-4:Tomcat9（Java11）を右クリック
6-5:追加及び削除をクリック
6-6:P2Av3をクリック
6-7:追加をクリック
6-8:完了をクリック


7.起動
7-1:eclipseの画面のP2Av3のフォルダ（プロジェクト）をクリック
7-2:srcをクリック
7-3:mainをクリック
7-4:webappをクリック
7-5:Top.jspを右クリック
7-6:選択の中の実行の中のサーバーで実行をクリック
7-7:起動
